# Campus Hub 📱🏫

Campus Hub is an ongoing mobile application built using *React Native* designed to streamline student life and keep everyone connected. It provides a central platform for authentication, real-time campus event tracking, announcements, and essential student utilities.

---

## 🚀 Features

* *Secure Authentication:* Simple login and signup functionality tailored for students.
* *Live Event Dashboard:* Real-time updates on upcoming campus events, seminars, sports, and social gatherings.
* *Smart Announcements:* A dedicated feed for instant campus notices and academic updates.
* *Interactive Campus Map (Planned):* Easy navigation across lecture halls, hostels, and student facilities.
* *Timetable Integration (Planned):* Personalized daily schedules for classes and exams.

---

## 🛠️ Tech Stack

* *Frontend:* React Native, Expo, TypeScript/JavaScript
* *Styling:* StyleSheet (Native components)
* *Version Control:* Git & GitHub
